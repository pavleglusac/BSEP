## Description

An application for managing your smart home in a secure manner!

## Getting Started

### Dependencies

* JDK 19
* npm 16
* mongodb running locally

### Executing program

To run spring backend, located in admin-be
* mvn spring-boot:run

To run both frontends, located in admin-fe and user-fe
* ng serve
